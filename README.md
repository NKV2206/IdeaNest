# IdeaNest
A Blogging Website
# How to set up Locally

## Features

- Sign Up Sign in functionality
- Display of all Blogs
- Page to display complete Blog
- Routing



## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
